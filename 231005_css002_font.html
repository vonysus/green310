font 속성

글꼴, 글자크기, 글자굵기, 기울임, 줄간격, 대소문자, 글자색상 등 7가지로 정리

색상을 제외한 여섯가지 속성은 한번에 작성 가능
속성 풀이
font-family 글꼴을 적용하는 속성으로 ''(따옴표)로 표기, serif(꺾임체)와 sans-serif(비꺾임체)로 구분
font-size 씨의 크기를 단위와 함께 작성 (ex. px, pt, %, em등으로 작성)
large, small과 같이 기준 단위보다 한단계 크고 작게
font-weight bold, normal로 표기 / 숫자 표기시 600, 400이 기준
font-style italic, normal로 표기 / 기울임체
font-variant small-caps, normal로 표기 / 작은 대문자로 표기
line-height 행간을 나타내고 단위가 1 -> 100% 기본값이 1.6임
font 위의 내용을 한줄에 표기가능
font {1번칸 2번칸 3번칸}
  • 1번칸 : font-weight font-style font-variants
  • 2번칸 : font-size/line-height 순서변경X line-height는 생략가능
  • 3번칸 : font-family
1번칸은 생략이 가능하나, 2번칸과 3번칸은 생략X, 순서 123 변경X

웹폰트 활용법 : 구글폰트에서 폰트를 선택 후 링크 or import를 먼저 복사해서 기입 후 font-family를 복사해서 적용

color

컬러는 글자의 색을 나타내며, background, border, shadow 등과 같이 컬러를 사용하는 공간에서 표현방식은 모두 같다.

  • 이름지정 : red, white, black 등
  • 핵사코드지정 : #000000(검정), #ffffff(흰색), 단 코드이름이 중복될 경우 3자리로도 가능 (ex. #000, #fff)
  • RGB 숫자지정 : rgb(0,0,0)검정, rgb(255, 255, 255)흰색
  • RGBA 숫자지정 : A - 투명도를 나타내는 alpha 값으로 0~1까지 소수점 표기
  • opacity : 투명도 지정한 모든 요소의 내부내용이 투명해진다.
