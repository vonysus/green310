박스속성_margin/padding

웹페이지의 레이아웃을 구성할 때 가장 중요한 요소
left / right / top / bottom으로 낱개로 설정가능

작성법

  • 한가지 숫자적용 - margin:10px 라고 작성시 4면 모두 동시적용
  • 두가지 숫자적용 - margin:10px 20px 라고 작성시 상하:10px 좌우:20px로 적용
  • 네가지 숫자적용 - margin: 5px 10px 15px 20px 라고 작성시 상:5px 우: 10px 하:15px 좌:20px로 적용
  • 건바이건으로 적용 - margin-bottom:10px

margin이란?

테두리와 다른 태그의 간격을 설정

  • 요소 1 / margin-bottom:10px
  • 요소 2 / margin-top:20px

padding이란?

요소의 내부 여백을 설정하여 공간으로 확보하는 형태

  • 요소 1
  • 요소 2

width / height

가로값 / 높이값으로 지정할 수 있으며 padding값과 border값의 영향을 받는다.

  • Lorem ipsum dolor sit amet consectetur
  • 요소2
  • Lorem ipsum dolor sit amet consectetur
  • 요소2

border란?

테두리의 모양, 색깔, 두께를 나타냄

  • 전체 테두리 : 5px 검정색 실선
  • 왼쪽선 : 20px 빨간색 실선
  • 윗쪽선 : 1px 파란색 점선
  • 오른쪽선 : 10px 파란색 두줄선
  • 아랫쪽선 : 2px 보라색 긴점선
